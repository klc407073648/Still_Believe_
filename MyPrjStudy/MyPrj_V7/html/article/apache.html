Apache的使用方法

1. Apache简介：

Apache程序是目前拥有很高市场占有率的Web服务程序之一，其跨平台和安全性广泛被认可且拥有快速、可靠、简单的API扩展

2.安装以及使用：

2.1安装以及开机自启

yum install httpd -y

systemctl start httpd

systemctl enable httpd

 

2.2配置服务器的访问地址：

修改apache的配置文件/etc/httpd/conf/ httpd.conf。

ServerRoot "/etc/httpd"

Listen 80   监听的IP地址与端口号

DocumentRoot "/var/www/html" 网站数据目录

DirectoryIndex index.html       默认的索引页页面

 

2.3 在DocumentRoot下创建first.html和index.html

// index.html代码为

<!DOCTYPE html>

<html lang="zh">

<head>

    <meta charset="UTF-8">

    <title>hello world</title>

</head>

<body>

Hello world！

</body>

2.4 访问网站

浏览器中输入：http://127.0.0.1:80/
